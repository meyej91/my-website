Once you're logged into GitHub, follow these steps to set up free hosting with GitHub Pages:

Step-by-Step Guide to Set Up GitHub Pages

1. Create a New Repository

  1. Click the + icon in the top-right corner of GitHub and select New repository.
  2. Name your repository (e.g., my-website).
  3. Add a description (optional).
  4. Set the repository to Public (required for GitHub Pages hosting).
  5. Check "Add a README file" for easier setup.
  6. Click Create repository.

2. Add Your Website Files

  • Option 1: Upload Files Manually

    1. On the repository page, click Add file > Upload files.
    2. Drag and drop your HTML, CSS, and JavaScript files (e.g., index.html, style.css, etc.).
    3. Commit the changes by clicking Commit changes at the bottom.

  • Option 2: Use GitHub Desktop

    1. Download and install GitHub Desktop.
    2. Clone your repository.
    3. Add your files locally and commit them using GitHub Desktop.

3. Enable GitHub Pages

  1. Go to your repository’s Settings tab.
  2. Scroll down to the "Pages" section (in the left-hand menu).
  3. Under Source, select the branch to publish from (usually main or master).
  4. Choose the root directory (default is /).
  5. Click Save.

4. Access Your Live Website

  • After a few seconds, GitHub will provide a URL to your live website (e.g., https://username.github.io/repository-name).
  • Test the URL in your browser. 🎉
